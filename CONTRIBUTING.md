# Contributing — BM Planning Site

Static landing page for BM Planning. **Product/code** changes go in the [bm-planning](../bm-planning) repo.

## Local

```bash
npm install
npm run dev
```

## What to contribute here

- Marketing copy (English)
- Pages `/features`, `/self-host`, `/roadmap`
- Accessibility and SEO
- GitHub Pages workflow (`.github/workflows/deploy.yml`)

## Checks

```bash
npm run build
```

## PRs

Use the template in `.github/pull_request_template.md`.

## Code of conduct

[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) — same spirit as the main repo.
